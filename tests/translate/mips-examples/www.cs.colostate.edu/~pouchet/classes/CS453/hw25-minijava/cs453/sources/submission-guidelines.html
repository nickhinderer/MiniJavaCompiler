CS 132 Project - Submission Guidelines

Your submission must work with the testing script when run on the SEAS Unix servers. If it does not, you will get a score of zero. Make sure you test your final tarball before submitting.

There is different testing script for each homework. Download the tarball, untar it, then read the instructions in the "ReadMe.txt" file.

  • hw2-tester.tgz
  • hw3-tester.tgz
  • hw4-tester.tgz
  • hw5-tester.tgz

NOTE: These grading script tarballs only contain a few test cases. Your submission will be graded against many more "private" test cases.
